# PeerConnect

PeerConnect is a Flutter & Dart-powered networking platform designed for students, mentors, and faculty to connect, collaborate, and grow.

## Features

### For Students:
- **Find Peers**: Discover peers based on filters such as department, skills, and interests.
- **Connect with Mentors & Faculty**: Build direct connections for guidance and support.
- **Upload Project Needs**: Share project requirements and get help from the community.
- **Engage in Discussions**: Participate in comment sections for collaborative discussions.
- **Stay Updated**: Access college news through integrated API feeds.

### For Mentors/Faculty:
- **Guide Students**: Provide mentorship via direct connections.
- **Profile Verification**: Implement 2-step authentication for secure profiles.
- **Boost Recognition**: Highlight student achievements and IDs for recognition.

## Why PeerConnect?
Unlike generic social apps, PeerConnect is tailored specifically for students and educators, offering:
- ✅ Secure networking with 2FA verification.
- ✅ Mentorship opportunities.
- ✅ Resource-sharing through uploads and comments.
- ✅ Campus news integration for staying informed.

## Getting Started

### Prerequisites
- Flutter SDK installed on your machine.
- Dart SDK (comes with Flutter).
- An IDE like VS Code or Android Studio.

### Installation
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/peerconnect.git
   ```
2. Navigate to the project directory:
   ```
   cd peerconnect
   ```
3. Install dependencies:
   ```
   flutter pub get
   ```
4. Run the app:
   ```
   flutter run
   ```

### Building for Production
- For Android: `flutter build apk`
- For iOS: `flutter build ios` (requires macOS)
